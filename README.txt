# 사용 방법

## 1. 설치
- 파이썬 설치  
> 다운로드 링크 : <https://www.python.org/ftp/python/3.10.3/python-3.10.3-amd64.exe>

- 필수 모듈 설치(plug in)  
  > pip install pandas openpyxl

---

## 2. 데이터 준비
- **원본 데이터 파일:**  `./in/지목변경신청서_통합.xlsx`  (해당 경로에 해당 파일명으로 엑셀 파일 준비)

  * 필수 시트명: `"1"`  (기존 시트명이 다르면 수정 후 실행 필수)

  * 필수 컬럼:  
    - 시군  
    - 읍면  
    - 동리  
    - 이동전지번  
    - 이동전지목  
    - 이동전면적  
    - 이동후지번  
    - 이동후지목  
    - 이동후면적

*(위 컬럼으로 구성된 엑셀 파일 준비)  

- **템플릿 파일:**  
  `./in/토지이동신청서_페이지.xlsx`  (해당 경로에 해당 파일명으로 엑셀 파일 준비)
  * 시트명: `"1", "2"`

---

## 3. 프로그램 실행

> python 토지이동신청서_지목변경.py

---

## 4. 결과 파일 확인
- 결과는 자동으로 다음 폴더에 저장됩니다(폴더 없을시 자동생성됨):  
  `./out`

- 파일명: 각 **행정구역명.xlsx** 형식으로 저장  
  (예: `ex)행정동.xlsx`, `ex)구룡동.xlsx` 등)

---

## 5. 주의사항
- 프로그램 실행 전, `./out_토지이동신청서_지목변경통합` 폴더에  
  이전 결과가 있다면 백업 후 삭제하는 것을 권장합니다.  
- `in` 폴더 내 파일명 또는 시트명이 다를 경우 오류가 발생할 수 있습니다.  
- Excel 파일은 반드시 **UTF-8 형식**으로 저장해야 합니다.  

---

## 7. 예시 실행 결과

```
✅ 완료: 12개 그룹 처리 → D:\프로젝트\out_토지이동신청서_지목변경통합
```
